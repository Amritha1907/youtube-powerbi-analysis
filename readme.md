# YouTube Video Performance Analysis

## Project Overview
This project analyses YouTube video performance data using Excel and Power BI. The dataset includes over 500 videos across multiple channels and categories, with metrics such as views, likes, publish date, and category ID.

## Objective
To clean and preprocess YouTube data using Excel and build an interactive Power BI dashboard to analyse channel performance, audience engagement, and publishing trends.

## Dataset
- Source: Public YouTube data
- Records: 500+ videos
- Domain: Social Media Analytics

## Tools Used
- Microsoft Excel – Data cleaning and preprocessing
- Power BI – Data modelling, DAX calculations, and dashboard creation

## Dashboard Highlights
- Total Views, Likes, Videos, and Channels
- Channel-wise performance analysis
- Category-wise video distribution
- Views and likes relationship analysis
- Time-based trend analysis
